# Feature Discovery Methodology

## The Discovery Process

1. **Receive Raw Input** - User provides unstructured feature idea
2. **Assess Scope** - Determine if single feature or epic
3. **Parallel Research** - 4 subagents research simultaneously
4. **Interactive Interview** - Ask questions until 100% clarity
5. **Generate Documentation** - Create FEATURE_DOCUMENTATION.md

## Research Subagents

### Subagent 1: Industry Best Practices
Search queries:
- "{feature} best practices 2025"
- "industry standard {feature} implementation"

### Subagent 2: Technical Patterns
Search queries:
- "{framework} {feature} implementation"
- "{feature} architecture patterns"

### Subagent 3: Competitive Analysis
Search queries:
- "popular apps {feature} examples"
- "{feature} UX patterns"

### Subagent 4: Security/Compliance
Search queries:
- "{feature} security considerations"
- "{data type} compliance requirements"
